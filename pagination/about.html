關於龍網

中華龍網


中華龍網股份有限公司 DragonSoft Security Associates, Inc.
成立於2003年，擁有優秀之專案團隊及安全專家，
致力於資訊安全軟體之研發，
以提供完整且價格合理之資訊安全解決方案為職志。

核心價值

中華龍網擁有優秀之研發團隊，以研發具有國際競爭力之資訊安全軟體為公司成立宗旨。 最具代表性之產品為自行研發之「弱點掃描軟體」( DVM, Dragonsoft Vulnerability Management )，DVM於2004年創下亞洲首先通過CVE認證（Common Vulnerabilities Exposures）的紀錄，並於2011年2月榮獲全球最大資訊安全雜誌SC Magazine最高榮譽五顆星滿分評比的肯定。 在台灣市佔率高達七成以上，客戶涵蓋政府機關、金融機構、學校、企業及資安服務廠商 ( SI ) 等二百多個單位。

附加價值

為提供台灣政府機關及企業優良之資安產品，不斷地提升資安技術及產品品質，並與國內外廠商合作交流資安防護，包括：最強大之專業DDoS防護設備Riorey； 美國知名防火牆及UTM大廠SonicWall；從矽谷回到台灣證券市場掛牌的知名資安公司「安瑞科技」( Array) 所生產之 SSL VPN及Load Balance設備；專業無線設備大廠 Arruba ； 快速竄起之美國矽谷SIEM廠商 Accelops ； VM Firewall 知名廠商 Reflex 等。 以客戶資安需求為優先，取經合作夥伴的長才，打造最安全的資安產品，提供客戶最適宜的資安服務。

未來展望

台灣政府於2010年6月通過「個人資料保護法」，未來政府及企業均將面對高標準之個人資料保護要求，若政府或企業不慎洩漏個資， 未來將可能面臨最高新台幣二億元之求償，企業負責人亦將負擔相關之民刑事責任。中華龍網為協助客戶因應個資法帶來之挑戰與衝擊， 特別於2010年成立個資安全中心，提供客戶「技術」面向之完整諮詢、技術查檢與顧問服務。
